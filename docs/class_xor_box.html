Detailed Description

A XorBox is a box that computes the bitwise xor of two inputs.

In order to simplify the implementation the the XorBox takes only one input in_bits but of size double of that of out_bits. the first half of in_bits represents the first of the two inputs, and the second half represents the last of the two inputs.
